System Overview

This Face Attendance System uses a multi-stage face recognition pipeline combining HOG (Histogram of Oriented Gradients) for face detection and a ResNet-based deep neural network for face encoding. The system achieves high accuracy while maintaining real-time performance for attendance marking.

Face Detection (HOG)
  • Converts image to grayscale
  • Calculates gradient magnitude and direction
  • Creates HOG feature vectors for image cells
  • SVM classifier determines face vs non-face
  • Non-maximum suppression removes overlapping detections
Advantage: Works on CPU, ~0.1s per detection
Face Encoding (ResNet)
  • Aligns face using eye landmarks
  • Passes through ResNet-34 architecture
  • Generates 128-dimensional embedding vector
  • Vector represents unique facial features
  • Similar faces have smaller Euclidean distance
Advantage: Creates robust, illumination-invariant features
Limitations & Best Practices
⚠️ Current Limitations:
  • Profile/angled faces may not be detected
  • Poor lighting reduces accuracy
  • Facial occlusions (glasses, masks) affect results
  • Only frontal faces optimal
✅ Best Practices:
  • Capture 3-5 photos from different angles during registration
  • Ensure good lighting when marking attendance
  • Face camera directly, maintain distance of 1-2 feet
  • Remove glasses/hats if possible during capture
